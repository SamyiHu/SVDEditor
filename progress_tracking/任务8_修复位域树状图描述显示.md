# 任务8：修复位域树状图描述显示（问题5）

## 问题描述
位域树状图的描述位置显示的是位宽，而不是描述。应该将描述显示在描述列，将位宽写到偏移列中。

## 根本原因
在`svd_tool/ui/components/peripheral_manager.py`中，位域子项的列设置有误：
- 树状图有5列（索引0-4）：名称、偏移、描述、访问权限、复位值
- 但位域子项的设置中，描述被设置到了第5列（索引5），而第2列（描述列）显示的是位宽
- 第5列不存在，导致描述无法显示

## 相关文件
- `svd_tool/ui/components/peripheral_manager.py`

## 修改内容

### 修改：位域子项列设置
**文件**: `svd_tool/ui/components/peripheral_manager.py`
**位置**: 第199-210行

**修改前**:
```python
# 添加位域子项
for field_name, field in register.fields.items():
    field_item = QTreeWidgetItem(reg_item)
    field_item.setText(0, field_name)
    field_item.setText(1, str(field.bit_offset))
    field_item.setText(2, str(field.bit_width))  # 错误：描述列显示位宽
    field_item.setText(3, field.access or "")
    field_item.setText(4, field.reset_value)
    field_item.setText(5, field.description)  # 错误：索引5不存在
    field_item.setData(0, Qt.ItemDataRole.UserRole, 'field')
    field_item.setData(0, Qt.ItemDataRole.UserRole + 1, field_name)
```

**修改后**:
```python
# 添加位域子项
for field_name, field in register.fields.items():
    field_item = QTreeWidgetItem(reg_item)
    field_item.setText(0, field_name)
    # 偏移列显示偏移和位宽（例如："0-3"表示从位0开始，宽度为3）
    field_item.setText(1, f"{field.bit_offset}-{field.bit_offset + field.bit_width}")
    # 描述列显示描述
    field_item.setText(2, field.description or "")
    field_item.setText(3, field.access or "")
    field_item.setText(4, field.reset_value)
    field_item.setData(0, Qt.ItemDataRole.UserRole, 'field')
    field_item.setData(0, Qt.ItemDataRole.UserRole + 1, field_name)
```

## 修复效果
- 描述列（第2列）现在正确显示位域的描述信息
- 偏移列（第1列）显示偏移和位宽信息，格式为"偏移-结束位"（例如："0-3"表示从位0开始，宽度为3）
- 移除了错误的第5列描述设置
